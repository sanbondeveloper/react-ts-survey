# Survey Form

## 프로젝트 소개
단답형, 장문형, 객관식, 체크박스, 드롭다운 5가지 형식의 질문을 제작할 수 있는 온라인 설문지 폼(Form)입니다.

- 개인 프로젝트
- 개발 기간: 약 1주
- 배포 주소: https://peppy-genie-f4960b.netlify.app/

### 주요 기능

**질문 추가 및 삭제**

질문을 추가하여 질문의 제목을 입력할 수 있고 삭제할 수 있습니다.
